Overview

The Rock, Paper, Scissors simulator is a project I developed as part of the Odin Project curriculum. It implements the classic game logic where players compete against a computer opponent, making selections of "rock," "paper," or "scissors." The program determines the winner based on the standard rules of the game and provides feedback on the results.
