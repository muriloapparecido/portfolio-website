Overview

The Etch-A-Sketch simulator is a project I developed as part of the Odin Project curriculum. It is an interactive web project that allows users to draw by hovering over a customizable grid. Inspired by the classic drawing toy, this project was built using HTML, CSS, and JavaScript to create a dynamic and engaging user experience.
