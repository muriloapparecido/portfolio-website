Overview

Welcome to my Personal Finance Tracker project! This is a simple tool designed to help users log and track their income and expenses. I am building this project as part of my journey to improve my web development skills.

Features (Current)

  • Static interface built using HTML and CSS.
  • Basic design layout to display income and expense categories.

Features (Planned)

  • Enable users to dynamically add, edit, and delete transactions.
  • Calculate and display total income, expenses, and balance.
  • Store transaction data in the browser using localStorage or sessionStorage.
  • Eventually integrate a backend database for long-term data storage.
  • Ensure mobile-friendly layouts using media queries.

Skills Used

  • HTML: Structured the webpage with semantic elements.
  • CSS: Styled the layout and design for better visual appeal.
